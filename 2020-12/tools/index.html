工具

发表于 2020-12-22 更新于 2023-07-12 分类于 timeline 阅读次数： 本文字数： 247 阅读时长 ≈ 1 分钟

Windows

7z：开源的，体积极小的解压/压缩软件  
Bandicam：录屏/屏幕录制工具，国区目前 249CNY/一台、376CNY/两台授权码
calibre：管理 epub、mobi、azw3 等格式的电子书工具，配合 kindle 使用极佳  
ChainLP：漫画制作工具，图片转 epub/mobi，kindle 可用，支持 apple book 观看  
everything：体积极小的文件搜索工具  
obs：开源的直播推流工具，可以录屏  
PotPlayer：本地视频播放器  
qBittorrent：磁力链接、BT 种子下载器  
ReNamer：批量重命名工具  
screentogif：gif 制作工具  
SETUNA：屏幕截图并置顶固定在桌面的工具  
spacesniffer：硬盘可视化为区块图的工具  
SumatraPDF：开源的体积极小的本地 pdf 阅读器  
VSCode：微软自家开源编码器，支持 markdown  
小丸工具箱：视频封装、压制工具，支持 264 和 265  

IOS

RdClient：远程连接 windows 工具
漫画收藏家：本地离线漫画阅读器，目前只支持图片序列
