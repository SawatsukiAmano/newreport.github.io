hexo-next7食用方法

Posted on 2020-12-21 Edited on 2021-01-11 In 博客搭建

博客启用（前略环境搭建）

&#8195;&#8195;新建文件夹，之后在其中执行：hexo init。
&#8195;&#8195;hexo-next7官方git：https://github.com/theme-next/hexo-theme-next
&#8195;&#8195;hexo-next8官方git：https://github.com/next-theme/hexo-theme-next
&#8195;&#8195;git配置：npm install –save hexo-deployer-git；多用户指定其一：githubgit config –global user.email “邮箱”和git config –global user.name “用户名”

可能会用到的工具

&#8195;&#8195;RSS订阅：npm install hexo-generator-feed；
参考：https://blog.csdn.net/qq_36537546/article/details/90730068
&#8195;&#8195;站点地图：npm install hexo-generator-sitemap –save；
参考：https://www.cnblogs.com/brady-wang/p/8493181.html
&#8195;&#8195;next7的搜索功能：npm install hexo-generator-searchdb –save；
参考：https://blog.csdn.net/qq_41086511/article/details/103537522
&#8195;&#8195;背景图片及透明度调整：\themes\next-7.7.2\source\css_common\components\pages\pages.styl，能找到此页面样式文件的均可。
参考：https://blog.csdn.net/Louis_li51/article/details/105227430；
注：带搜索框后不能开启菜单栏和搜索栏的透明度，否则会透明度过高看不清，但可以给头像的框加透明度。
&#8195;&#8195;next7单独友链参考：https://blog.csdn.net/Aoman_Hao/article/details/103758829
&#8195;&#8195;文章置顶：node_modules/hexo-generator-index/lib/generator；
参考：https://www.jianshu.com/p/42a4efcdf8d7
&#8195;&#8195;图片压缩工具：https://tinypng.com/ 或者https://docsmall.com/image-compress
&#8195;&#8195;图片批量重命名工具（可为图片hash值）：ReNamer
