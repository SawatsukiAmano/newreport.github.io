> 系统为 Rocky Linux，podman 已设置别名为 docker
```bash
# 编辑开机任务
crontab -e
# 设置开机自启
@reboot docker ps -a |awk '{print $1}'|grep -v 'CONTAINER'|while read line;do docker start "$line";done
```
